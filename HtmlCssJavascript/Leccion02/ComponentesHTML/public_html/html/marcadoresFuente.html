Marcadores de fuente

B: se utiliza para poner en negritas el texto

BIG: se utiliza para poner el texto en una fuente más grande

I: se utiliza para poner el texto en cursiva

S: se utiliza para poner una raya en el texto

SMALL: se utiliza para poner el texto más pequeño

STRIKE: el mismo resultado que s, poner una raya en el texto

TT: se utiliza para poner el texto como se solía utilizar en impresoras y monitores antiguos

U: se utiliza para subrayar el texto
